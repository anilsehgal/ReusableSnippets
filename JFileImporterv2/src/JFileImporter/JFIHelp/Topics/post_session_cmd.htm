Post Session Command

Post Session Command is an advanced feature given if some command is required to be run after the actual import completes. Post session command is of two types:

  • SQL Command: This runs any mentioned sql on the database mentioned in the DataBase Url Box
    Example Usage: To update a particular column of all rows after an import completes
  • System Command: This runs any mentioned DOS CMD on the system
    Example Usage: Any FTP command in DOS or any file copy or deletion command
