Pre Session Command

Pre Session Command is an advanced feature given if some command is required to be run before the actual import begins. Pre session command is of two types:

  • SQL Command: This runs any mentioned sql on the database mentioned in the DataBase Url Box
    Example Usage: To Clear Out existing Data before Import
  • System Command: This runs any mentioned DOS CMD on the system
    Example Usage: Any FTP command in DOS or any file copy or deletion command
