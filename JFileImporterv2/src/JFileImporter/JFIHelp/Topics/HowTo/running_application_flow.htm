Running Importer Application

Use the following steps to run the import process:

  1. select a File Flavor from the File Settings Tab
  2. Ensure that you have either the Delimiter or Excel Sheet No. or Fixed Length Column Descriptor Table typed in
  3. click the Open File Button and select a file from the desired path
  4. Click on open or Double Click the file to be opened
  5. To view the opened file, click on the File Display tab, the file as read in Table Format as per the given conditions is displayed there
  6. Go to the DB Settings tab and enter the required fields to fetch database tables
  7. Fetch all the DB tables by clicking the Fetch Tables button
  8. Select a table from the Tables fetched, all the columns of that table should be populated in the Table Column Mapper Schema
  9. Map the DB Columns to the File Columns as per the data in File Display tab
  10. Select a preferred Duplicate Primary Key Action
  11. Click on Validate Import for file field datatype validation against DB columns
  12. After the Start Import button is enabled, click on it to process import
