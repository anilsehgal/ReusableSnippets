DB/File Column Mapper

DB/File Column Mapper contains following information about all the columns of the table selected from the Tables List.

  • Column Name: This contains the DB Column Names for all the columns of the selected table
  • Column Datatype: This contains the DB Column datatype for all the columns of the selected table
  • Column length: This contains the DB Column field length for all the columns of the selected table
  • Column Nullability: This contains the information about DB Column nullability for all the columns of the selected table
  • Column is Primary Key: This contains the information whether the DB column is a part of the primary key, for all the columns of the selected table
  • File Column: This contains a drop menu with all the file column headers, for all the columns of the selected table
