Date Format

Date Format is a String field which defines the Date Format of the input file date columns. Following fields collate a date format:

  • yy: Last two digits of the year
  • yyyy: four digits of the year
  • mm: Last two digits of the month
  • dd: two digits of the day
  • hh: two digits of the hour
  • MM: two digits of the minutes
  • ss: two digits of the seconds

If no date format is provided, the default date format taken is: yyyy-mm-dd
