Running Scheduler Application

Use the following steps to start the schedule:

  1. Click on Open Profile button from the scheduler application
  2. Browse for the saved profile which is to be opened, only profiles saved by the Save Profile button can be opened
  3. Once opened, the profile details are loaded in the Column Mapping and Import Options tabs, these are view only tabs
  4. Now, go to Schedule Settings tab and select a Schedule Start Date, Schedule End Date and define a repeat interval
  5. Click on the Schedule Start button to start the schedule
